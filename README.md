# Beginner Chess App (starter)

Prereqs:
- Node.js 18+ recommended
- npm or yarn

Install:
npm install

Run (dev):
npm run dev

Then open the URL printed by Vite (usually http://localhost:5173).

Features:
- Click a piece (when it's your turn), then click a destination
- Lessons in the right panel
- Simple AI (easy/medium/hard)
- Save progress automatically to localStorage
- Export PGN button copies the current game PGN to clipboard
